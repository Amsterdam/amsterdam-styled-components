# breakpoint

A function to retrieve a breakpoint value from a theme

- mobileS: 320px
- mobileM: 375px
- mobileL: 414px
- tabletS: 540px
- tabletM: 768px
- laptop: 1024px
- laptopM: 1200px
- laptopL: 1430px
- desktop: 1920px
- desktopL: 2560px

## Min-with, laptopM
