# breakpoint

A function to retrieve the breakpoint value from a theme. The default theme only has one breakpoint @ 820px.

## Default

<Canvas>
  <Story name="Default">
    {(() => {
      const MyDivWithThemeBreakpointDiv = styled.div`
        @media screen and ${breakpoint()} {
          display: none;
        }
      `
      return (
        <Fragment>
          Component below is only visible below 820px and will print: I am
          visible
          <MyDivWithThemeBreakpointDiv>
            I am visible
          </MyDivWithThemeBreakpointDiv>
        </Fragment>
      )
    })()}
  </Story>
</Canvas>
