# useTrappedFocus hook

A hook that will trap the focus, useful in Modals

_Note:_ these only work in the "Canvas" view

## Default

<Preview>
  <Story name="Default">
    <Fragment>
      {(() => {
        const ref = useRef()
        const { keyDown } = useTrappedFocus(ref)
        return (
          <Fragment>
            <Button>Outside button</Button>
            <div role="presentation" onKeyDown={keyDown} ref={ref}>
              <p>
                Once focussed in here, you can not focus on elements outside
              </p>
              <Button>One</Button>
              <Button>Two</Button>
            </div>
            <Button>Outside button</Button>
          </Fragment>
        )
      })()}
    </Fragment>
  </Story>
</Preview>
