# useTrappedFocus hook

A hook that will trap the focus, useful in Modals

## Default

<Preview>
  <Story name="Default">
    <React.Fragment>
      {(() => {
        const ref = useRef()
        const { keyDown } = hooks.useTrappedFocus(ref)
        return (
          <React.Fragment>
            <Button>Outside button</Button>
            <div role="presentation" onKeyDown={keyDown} ref={ref}>
              <p>
                Once focussed in here, you can not focus on elements outside
              </p>
              <Button>One</Button>
              <Button>Two</Button>
            </div>
            <Button>Outside button</Button>
          </React.Fragment>
        )
      })()}
    </React.Fragment>
  </Story>
</Preview>
