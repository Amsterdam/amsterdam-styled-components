# themeSpacing

A function to retrieve a value derived from the theme spacing unit

## Default

<Preview>
  <Story name="Default">
    <Fragment>
      {(() => {
        const MyThemeSpacedDiv = styled.div`
          color: ${themeColor('tint', 'level1')};
          background-color: ${themeColor('primary')};
          padding: ${themeSpacing(2)};
        `
        return (
          <MyThemeSpacedDiv>
            I have a padding of 8px (themeSpacing(2), as our spacing unit is
            4px, so 2 x 4 = 8px)
          </MyThemeSpacedDiv>
        )
      })()}
    </Fragment>
  </Story>
