# themeSpacing

A function to retrieve a value derived from the theme spacing unit

## Default

<Canvas>
  <Story name="Default">
    <Fragment>
      {(() => {
        const MyThemeSpacedDiv = styled.div`
          color: ${themeColor('tint', 'level1')};
          background-color: ${themeColor('primary')};
          padding: ${themeSpacing(2)};
        `
        return (
          <MyThemeSpacedDiv>
            I have a padding of 16px (themeSpacing(2), as our spacing unit is
            8px, so 2 x 8 = 16px)
          </MyThemeSpacedDiv>
        )
      })()}
    </Fragment>
  </Story>
</Canvas>
