# Pagination

[Design System](https://designsystem.amsterdam.nl/7awj1hc9f/p/431c4a-pagination)

**Component status: experimental**

The Design System team has placed an item on their backlog to decide on how this pagination bar should behave on
smaller viewports. With the current spacing the pagination bar does not fit on for example 320px.
