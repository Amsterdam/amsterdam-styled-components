# StepByStepNav

[Design System](https://designsystem.amsterdam.nl/7awj1hc9f/p/0659fc-step-by-step-navigation)

**Component status: experimental**

Step by step navigation component. Does not contains links. Can displays three different variants: no visual indicators other than the blue and gray discs, numerical indicators and check-mark indicators. Will not render in a viewport that is configured to have a width that is equal to or smaller than the `mobileS` breakpoint (defaults to 320 pixels).

## Checkmark

Will show a checkmark icon on each of the items that are before the active item. For all items to get a checkmark, set the `stepsCompleted` prop to 'true'.
