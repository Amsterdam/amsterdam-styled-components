# Header

**Component status: experimental**

## Short only

Given the `tall={false}` prop, this will make sure the header is not showing its tall variant after breakpoint laptopM (1200px)

<Canvas>
  <Story name="Short only">
    <Fragment>
      <Header tall={false} title="Some title" fullWidth={false} />
    </Fragment>
  </Story>
</Canvas>

# Full width

<Canvas>

    </Fragment>
  </Story>
</Canvas>

# Tall

To see the tall version, please make sure your viewport is bigger than laptopM (1200px)

<Canvas>
  <Story name="Tall">
    <Fragment>
      <Header tall title="Some title" fullWidth={false} />
    </Fragment>
  </Story>
</Canvas>

## Overwrite the default h1 tag

Use `headerLogoTextAs` to overwrite the default `h1` tag on the logo. Useful for SEO purposes.

<Canvas>
  <Story name="Overwrite the default h1 tag">
    <Fragment>
      <Header headerLogoTextAs="div" tall={false} title="Some title" />
    </Fragment>
  </Story>
</Canvas>
