# CustomHTMLBlock

This component will apply styles from Heading, Paragraph, Link, Blockquote and List on non-styled HTML
This is useful when you want to style HTML coming from a CMS.

**Component status: Experimental**

## HTML passed as a child

<Canvas>
  <Story name="HTML passed as a child">
    <CustomHTMLBlock>
      <h1>Hello, I am a styled h1</h1>
      <p>And I am a styled paragraph, we are passed as child components</p>
      <ul>
        <li>Unordered list item 1</li>
        <li>Unordered list item 2</li>
        <li>Unordered list item 3</li>
      </ul>
      <ol>
        <li>Ordered list 1</li>
        <li>Ordered list 2</li>
        <li>Ordered list 3</li>
      </ol>
      <a href="/">I am a link</a>
      <blockquote>I am a blockquote</blockquote>
    </CustomHTMLBlock>
  </Story>
</Canvas>
